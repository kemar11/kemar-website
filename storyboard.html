Purpose of Each Webpage

  • Home: Introduces visitors to the portfolio.
  • Professional: Showcases resume professional projects and work experiences.
  • Personality: Provides insights into personal traits, role model and interests.
  • Personal Development: Highlights ongoing personal growth and skills
  • Storyboard: Displays planning related to the portfolio design.
  • References: Provides references which were used to complete the website.

Theme of the Site

The website follows a clean,simple theme with a focus on readability. It uses the following design elements:

  • Color Scheme: Predominantly blue and white. Blue is used for headers, navigation, and links, while white serves as the background for the content.
  • Font: Simple, readable fonts for body text, and larger, bold fonts for headings.
  • Graphics: Minimalistic icons and images to keep the layout clean and focused on content.

Website Features

The following were implemented in the Website:

  • Contact Form: A contact form allowing visitors to reach out via email.
  • Navigation Bar: A navigation bar allowing users to access different sections of the webpage.
